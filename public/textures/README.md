# Web Textures

Dieser Ordner ist fuer optimierte Laufzeit-Texturen, die von der Website
geladen werden duerfen.

Empfohlene Struktur:

- `public/textures/<asset-name>/...`

Geeignete Formate:

- `.jpg`
- `.png`
- `.webp`
- `.ktx2`

Nicht hier ablegen:

- unbearbeitete 8K-Quelltexturen
- Photoshop-Dateien
- Blender-Quellen
- Referenzbilder
